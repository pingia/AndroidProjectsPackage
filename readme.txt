1. Unzip the three files AndroidPkg.ps1, package.bat, and readme.txt in the compressed package to the English directory,
2. Open the android directory, execute package.bat to start packaging
2. Follow the interface prompts to operate,
3. When the packaging is complete, a dialog box will pop up, select "Copy Package Path"
4. Paste the path in the windows browser and jump to the packaging directory
5. Copy and package apk or aab
